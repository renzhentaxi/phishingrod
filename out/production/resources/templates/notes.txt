requirement to be a pr-node
1. must contain a text node or is anchor.
2. if has multiple text node, each text node is split into their own pr-node
3. if has only 1 text node,then the parent node is part of a pr-node

pr-node-type: text
    div element
    wraps around a text node.

pr-node-type:link
    an anchor element

pr-node-type: link-text
    div element
    text of an anchor element

pr-node-type: single
    can be any element
